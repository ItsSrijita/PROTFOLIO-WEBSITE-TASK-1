## 📖 About This Project

This is a **fully responsive personal portfolio website** built from scratch using **HTML5, CSS3, and JavaScript**. It showcases my skills, projects, education, and contact information in a clean, modern, and accessible design.

The website is designed to serve as my **professional online presence** for recruiters, hiring managers, and collaborators. It follows **modern web design principles** with a focus on:

- ✅ **Mobile-first responsiveness** – works flawlessly on phones, tablets, and desktops.
- ✅ **Accessibility** – semantic HTML5, ARIA labels, and keyboard navigation.
- ✅ **Performance** – lightweight, fast-loading, no external dependencies besides fonts and icons.
- ✅ **Modern UI** – gradient accents, smooth transitions, and a no‑black color palette.

## ✨ Features

### 🎨 Design Highlights

- **No black anywhere** – all colors are warm, vibrant, and accessible.
- **Gradient accents** – primary purple (#6C63FF) and secondary pink (#FF6584).
- **Glass‑morphism header** – blurred background with subtle shadow.
- **Floating profile avatar** – gentle animation with hover scale.
- **Smooth hover effects** – buttons, cards, and links all have polished transitions.

### 📱 Responsive Breakpoints

| Breakpoint | Target Devices |
|------------|----------------|
| `≥ 1024px` | Desktops & large screens |
| `768px – 1023px` | Tablets & small laptops |
| `480px – 767px` | Mobile phones (landscape) |
| `< 480px` | Small phones (portrait) |

### ♿ Accessibility

- Semantic HTML5 elements: `<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`, `<article>`.
- ARIA labels: `aria-label`, `aria-hidden`, `aria-expanded`, `role="menubar"`, etc.
- Keyboard‑friendly navigation.
- Form validation with visual error messages.
- Focus states for all interactive elements.
